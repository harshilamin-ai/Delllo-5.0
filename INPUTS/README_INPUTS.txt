Put the latest CILGeneration files here before running RAIN5.0:

Required:
- CILProfiles_<ddmmyyhhmm>.csv

Optional but recommended for calibration:
- matchesCreated_<ddmmyyhhmm>.csv

RAIN5.0 uses CILProfiles as the prediction universe. matchesCreated is treated as labelled training/calibration data and is copied into each OUTPUT run folder.
